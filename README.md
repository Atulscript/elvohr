# React + Vite Project

This project is built using **React** and **Vite**, providing a fast and modern development environment with Hot Module Replacement (HMR) and optimized production builds.

## Prerequisites

- Node.js 18 or later
- npm or yarn

## Installation

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Production Build

Create an optimized production build:

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React
- Vite
- JavaScript
- HTML5
- CSS3
